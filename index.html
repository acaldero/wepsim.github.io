Description

    WepSIM is a software simulation tool for teaching Computer Structure with three main goals:
    • WepSIM offers an integrated vision of microprogramming and assembly programming while different instruction sets can be defined and used in the assembly code.
    • WepSIM gives more flexibility to the students because it can be used by a wide range of devices (including smartphones and tablets). For example, WepSIM allows the students to learn while travelling by subway. % It is valuable complement of a good exercises book.
    • WepSIM was created with a modular design, so it is possible to add, remove or modify the existing elements.
