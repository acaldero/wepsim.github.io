WepSIM
Web Elemental Processor SIMulator

Description

WepSIM is a software simulation tool for teaching Computer Structure.
It is a very valuable complement of a good exercises book, because students can easily check "What will happen if...".

WepSIM offers many advantages compared to traditional tools, among them:

Install

Quick Start

An integrated vision of microprogramming and assembly programming

where different instruction sets can be defined and used in the assembly code.

More flexibility to the students to practice

because it can be used by a wide range of devices (including smartphones and tablets).
For example, WepSIM allows the students to learn while travelling by subway.

A flexible simulation

because WepSIM was created with a modular design, so it is possible to add, remove or modify the existing elements.
